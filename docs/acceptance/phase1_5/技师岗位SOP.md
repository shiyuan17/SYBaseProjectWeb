# 技师岗位 SOP

## 适用范围

- 角色定位：病理科接收技师、取材技师、脱水/包埋/切片/染色技师
- 试点范围：`M2` 标本接收与 `M3` 技术工作流主链路，联动 `M5` 归档、试剂、设备异常上报
- 本轮目标：保证病例从接收到出片全程可追踪，并在归档、试剂、设备问题出现时完成标准化交接

## 演练前检查

- 账号具备标本接收、技术任务查询、对应工序执行权限
- 试剂库存、设备保养提醒、归档借阅联系人名单已确认
- 扫码终端、标签设备、显微镜或相关工位可正常使用

## 主流程

1. 在登记接收工作站完成标本接收，核对申请单、标本袋、容器数量和接收时间。
2. 按任务顺序进入取材、脱水、包埋、切片、染色等工作站，逐步完成状态流转并记录操作人。
3. 对需要返工、补材、医嘱执行或特殊处理的病例，按系统提示登记原因和处理措施。
4. 若医生在工作台反馈需要追查蜡块、玻片或申请单归档位置，先核对本环节编号是否完整，再联系归档管理员执行借阅或查找。
5. 若演练中发现试剂低库存、近效期或设备到期提醒，立即通知试剂设备管理员确认是否影响当班作业。
6. 完成当班任务后复核病例链路、玻片数量、返工记录和异常闭环情况。

## 异常处理

| 场景 | 处理动作 | 验收口径 |
| --- | --- | --- |
| 标本接收信息不完整 | 先挂起接收，联系护士补齐离体/固定/交接信息 | 不允许带缺失关键信息进入技术链 |
| 取材或包埋数量与系统不符 | 立即暂停流转，核对包埋盒、玻片、备注和返工单 | 需形成差异说明后再继续 |
| 医嘱要求重切/补取 | 在对应工作站登记处理结果，并同步医生 | 医嘱状态必须可追溯 |
| 发现试剂库存不足 | 停止继续消耗高风险试剂，通知试剂设备管理员 | 问题需登记到台账并标注影响批次 |
| 发现设备保养逾期 | 不再安排高风险样本上机，联系设备管理员评估 | 若继续使用，必须有人工审批记录 |

## 权限缺失

- 表现：无法进入技术工作站、无法开始/完成工序、无法查看任务列表
- 自查顺序：
  1. 确认当前账号是否为技师岗位账号
  2. 检查是否被错误授予护士或医生角色
  3. 由管理员核对 `M3` 工序权限和任务查询权限
- 临时策略：
  - 不共用他人账号
  - 由当班负责人在纸质流转单上记录工序开始/结束时间
  - 班后由授权账号按原始记录补录，并在问题台账标记权限缺失

## 人工兜底

- 工位终端不可用：切换备用终端；仍失败时用纸质流转卡记录
- 试剂或设备异常：优先保证在制样本安全，改走人工分批或暂停上机
- 归档位置无法确认：由医生或技师提供病例号、蜡块号、玻片号，归档管理员人工检索

## 复核记录

- 复核人：工序组长或当班质控技师
- 复核时点：接收完成后、切片出片前、当班结束前
- 复核内容：
  - 病例号、包埋盒号、玻片号链路一致
  - 超时、返工、退回、补材原因完整
  - 试剂/设备异常已通知对应管理员
  - 需要归档交接的材料编号完整
- 复核留痕模板：

| 日期       | 病例号 | 当前工序 | 复核结果      | 风险说明 | 复核人 |
| ---------- | ------ | -------- | ------------- | -------- | ------ |
| YYYY-MM-DD |        |          | 通过 / 不通过 |          |        |

## 升级联系人

- 一级：当班技术组长
- 二级：病理科质量负责人
- 三级：归档管理员 / 试剂设备管理员
- 四级：Phase 1.5 前端联调负责人 / 后端接口负责人
