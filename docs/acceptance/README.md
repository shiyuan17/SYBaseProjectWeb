# Acceptance 文档索引

## 说明

本目录用于归档验收、交付和现场执行材料，重点服务于联调验收、发布收口和角色演练。

## 文档清单

- [full-process-acceptance-record-template.md](./full-process-acceptance-record-template.md)
- [full-process-pr-description.md](./full-process-pr-description.md)
- [full-process-role-accessibility-checklist.md](./full-process-role-accessibility-checklist.md)
- [M1_M4_FULL_TEST_HANDBOOK.html](./M1_M4_FULL_TEST_HANDBOOK.html)
- [M1_M6_BUSINESS_PROCESS_GUIDE.html](./M1_M6_BUSINESS_PROCESS_GUIDE.html)
- [phase1_5/README.md](./phase1_5/README.md)

## 推荐阅读顺序

1. 先看 [full-process-role-accessibility-checklist.md](./full-process-role-accessibility-checklist.md) 了解角色覆盖范围。
2. 再看 [full-process-pr-description.md](./full-process-pr-description.md) 把握交付边界与验证口径。
3. 进入 [full-process-acceptance-record-template.md](./full-process-acceptance-record-template.md) 执行目标环境验收。
4. 需要详细操作说明时，再查看测试手册、业务手册和 `phase1_5` 岗位 SOP。

## 与其他目录的关系

- 依赖 [../plans/README.md](../plans/README.md) 提供实施背景与范围说明。
- 依赖 [../reviews/README.md](../reviews/README.md) 提供审查和差距分析结论。
- 可结合 [../reference/README.md](../reference/README.md) 补充业务模块与数据库背景。

## 维护约定

- 验收模板、PR 说明、角色清单、现场 SOP 一律归档到本目录。
- 新增现场演练资料时，优先放在本目录下的专题子目录，而不是重新放回 `docs/` 顶层。
