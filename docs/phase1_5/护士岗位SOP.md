# 护士岗位 SOP

## 适用范围

- 角色定位：手术室护士、标本固定核对护士、转运交接护士
- 试点范围：`M2` 标本采集登记、固定核对、转运交接，以及向 `M4/M5` 下游提供完整交接信息
- 本轮目标：确保送检链路数据完整、标签一致、交接可追溯，为后续医生归档回流和 M5 现场演练提供稳定前置数据

## 演练前检查

- 账号已启用，具备标本登记、固定核对、转运交接、轨迹查询权限
- 标签打印机、扫码枪、病区网络可用
- 当班交接本、纸质应急登记单、联系电话清单已到位

## 主流程

1. 进入病理申请或标本采集登记页面，核对患者身份、申请单号、送检类型和标本数量。
2. 扫码或手工录入标本信息，打印申请单标签与标本标签，双人核对条码、病人姓名和标本名称。
3. 在固定核对环节补录离体时间、固定开始时间和固定液信息，确认未固定提醒已清零。
4. 创建转运单并完成交接，确保转运单号、交接人、时间戳和标本件数一致。
5. 通知病理科接收人员，必要时在轨迹页面确认标本已进入“待接收”或“已接收”状态。
6. 若医生或管理员反馈归档回流异常，优先核对本环节申请单号、条码和交接时间是否准确，再提交升级。

## 异常处理

| 场景 | 处理动作 | 验收口径 |
| --- | --- | --- |
| 标签打印失败 | 先暂停交接，改为重新打印；仍失败时启用纸质备用标签并手写申请单号 | 备用标签必须有双人签名，后续补录系统 |
| 扫码无法识别 | 改用申请单号或条码手输，并核对病人姓名与标本名称 | 不允许跳过身份核对直接送出 |
| 固定时间遗漏 | 立即补录实际时间并在备注中说明原因 | 现场演练记录需保留补录说明 |
| 转运单件数不一致 | 停止交接，现场复盘件数与容器数量 | 未闭环前不得继续下送 |
| 病理科拒收或部分接收 | 按拒收原因回传手术间并记录在交接本 | 需保留拒收原因和责任回路 |

## 权限缺失

- 表现：页面入口不可见、扫码后提示无权限、无法创建转运单或查询轨迹
- 自查顺序：
  1. 重新登录并确认当前岗位账号
  2. 核对是否误用医生、技师或管理员账号
  3. 由当班管理员核对角色是否包含 `M2` 相关权限
- 临时策略：
  - 不借用他人账号操作
  - 改走纸质登记 + 电话报备 + 班后补录
  - 在问题台账中登记“权限缺失”并附申请单号

## 人工兜底

- 病人信息核对失败：使用纸质送检单完成封包，由当班护士长签字后移交
- 网络中断：保留纸质申请单、拍照留存标签和容器信息，网络恢复后 30 分钟内补录
- 下游无法查询到轨迹：电话通知病理科接收岗，按申请单号人工查找并记录处理时间

## 复核记录

- 复核人：当班第二护士或护士长
- 复核时点：标签打印后、固定完成后、转运交接前
- 复核内容：
  - 申请单号与标签一致
  - 标本数量与容器数量一致
  - 离体/固定/交接时间完整
  - 异常是否已登记与回传
- 复核留痕模板：

| 日期       | 申请单号 | 标本数 | 复核结果      | 异常说明 | 复核人 |
| ---------- | -------- | ------ | ------------- | -------- | ------ |
| YYYY-MM-DD |          |        | 通过 / 不通过 |          |        |

## 升级联系人

- 一级：手术室当班护士长
- 二级：病理科接收组负责人
- 三级：病理系统管理员
- 四级：Phase 1.5 前端联调负责人 / 后端接口负责人
