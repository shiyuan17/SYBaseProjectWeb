# 临床标本送检模块

## 模块定位

本模块用于管理临床手术标本送检流程，辅助临床医生完成标本登记、标本核对、标本转运等过程，通过二维码实现对标本的数字化标识，解决标本离体时间、固定时间的准确记录，以及标本名称的规范化命名。

## 子功能清单

### 病理申请工作站

- 功能说明: 通过与其他第三方系统对接，拉取患者信息并自动填写申请单，支持打印申请单。
- 关键补充: 自动根据关键属性检索患者已存在的申请单，并对操作人员进行提示，减少工作量和失误。

### 标本采集登记管理

- 功能说明: 用于手术台护士辅助手术医生完成标本采集及标本标签打印。
- 关键补充: 支持扫描条码或二维码获取申请单信息并校验。
- 关键补充: 支持多模式录入标本，灵活打印标本及申请单标签。
- 关键补充: 可根据不同业务或送检类型，按电脑 IP 设置不同标签模板，适配不同型号标签打印机。
- 关键补充: 支持手术医生查看冰冻检查进度，并查看标本轨迹与各阶段详情。

### 标本固定核对管理

- 功能说明: 用于辅助台下护士完成添加固定液及标本核对。
- 关键补充: 通过扫描标本二维码完成核对并自动记录固定时间。
- 关键补充: 提供未固定标本数量提醒功能，避免遗漏。

### 标本转运管理

- 功能说明: 用于手术室护士与转运工人之间的标本交接。
- 关键补充: 支持创建转运单、标本校验、转运单状态存储与打印。
- 关键补充: 支持查询待运单及未固定标本提醒。

### 标本接收管理

- 功能说明: 用于病理科接收标本，提供接收查询、扫码核对、未收提醒等能力。
- 关键补充: 支持拒收或部分接收，并支持质量评价。
- 关键补充: 支持未接收标本数量提醒，避免接收遗漏。
