# 集成与历史数据

## 模块定位

本部分用于整理系统建设中的支撑能力，包括第三方硬件设备接口集成、第三方系统软件接口集成和历史数据处理，侧重数据交互、设备接入和历史报告迁移。

## 子功能清单

### 第三方硬件设备接口集成

- 功能说明: 为更好建设数字化、智慧化系统，在具备接入条件时将医院已有硬件设备接入系统软件。
- 关键补充: 涉及硬件接口且需要设备厂商提供 SDK 的，需要医院协调。

### 第三方系统软件接口集成

- 功能说明: 系统建设除自身功能外，还需要与其他相关系统软件进行数据交互和接口通讯。
- 关键补充: 涉及软件接口且需要厂商提供接口的，需要医院协调。

### 历史数据处理

- 功能说明: 由医院协调原有系统提供报告数据接口，新建系统实现将原有系统的报告数据导入新系统。
- 关键补充: 目标是在新系统中查询历史报告数据。
