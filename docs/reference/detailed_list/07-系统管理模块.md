# 系统管理模块

## 模块定位

系统软件作为科室全流程质控管理的核心信息系统，必须具备全面的系统管理功能，并通过系统管理扩展适应具体业务使用流程。

## 子功能清单

### 系统用户管理

- 功能说明: 用于在系统用户管理工作站中检索和展示当前系统用户基本信息。
- 关键补充: 检索结果支持 Excel 导出和 Excel 批量导入。
- 关键补充: 支持编辑用户角色、工号、职称等信息。
- 关键补充: 支持记录并显示账户最近使用情况，便于排查账号问题。
- 关键补充: 提供标签登录功能，并支持直接打印用户登录标签，适配病理科不同登录场景。

### 用户角色管理

- 功能说明: 提供角色权限模型，通过先为角色赋权、再将角色赋予账户的方式管理权限。
- 关键补充: 支持角色新增、删除和角色授权。
- 关键补充: 角色授权内容包括工作模块或子系统、角色权限、消息订阅和角色统计类型授权等。

### 描写模板管理

- 功能说明: 提供描写模板管理工作站，以树形结构形式管理描写模板。
- 关键补充: 支持用户自行新建树形目录，并对目录进行新增、修改和删除。
- 关键补充: 支持录入模板文字内容，便于一键加入描写内容。
- 关键补充: 支持模板切分部位数量配置，适配自动生成包埋盒过程。
- 关键补充: 支持模板适用部位配置，适配智能匹配取材描写模板。
- 关键补充: 支持模板复制、修改、删除和条件检索等基础功能。

### 取材规范管理

- 功能说明: 提供取材规范库管理功能，通过树形结构展示规范内容。
- 关键补充: 支持用户自定义新增、删除和修改规范目录节点。
- 关键补充: 提供复制、剪切、插入图片、插入表格、对齐、加粗、字号调整等丰富编辑工具。
- 关键补充: 用于实现取材规范知识图谱的高度自定义编辑。
- 关键补充: 支持规范内容的新增、修改、删除和检索等基础功能。

### 部位字典管理

- 功能说明: 提供部位字典管理工作站，以树形结构展示部位字典信息。
- 关键补充: 支持部位新增、修改、删除、检索和编辑。

### 检查号规则管理

- 功能说明: 提供检查号规则管理工作站，以列表形式展示当前系统已配置的检查号规则。
- 关键补充: 支持按不同送检类型自定义规则，自定义内容包括固定前缀和周期性自增部分。
- 关键补充: 周期性自增可按年等周期设定。
- 关键补充: 支持规则新增、编辑和删除等基础操作。

### 医嘱字典管理

- 功能说明: 提供医嘱字典管理工作站，以树形结构展示当前医嘱字典。
- 关键补充: 支持用户自定义管理分类。
- 关键补充: 支持医嘱字典新增、修改、删除和检索等基础功能。
- 关键补充: 支持字典停用和重新启用。

### 医嘱套餐管理

- 功能说明: 提供医嘱套餐管理工作站，以树形结构展示当前医嘱套餐信息。
- 关键补充: 支持新增、修改和删除套餐目录。
- 关键补充: 对常用医嘱套餐可通过勾选医嘱项目进行管理，无需手动文字输入。
- 关键补充: 提供私有套餐和特检套餐管理。

### 医嘱收费管理

- 功能说明: 提供医嘱收费管理工作站，以树形结构展示当前医嘱收费信息。
- 关键补充: 支持双击打开收费字典信息并编辑对应收费项。
- 关键补充: 支持医嘱收费项目新增、修改、删除。
- 关键补充: 支持导出医嘱收费项目和批量导入医嘱收费项目。

### 配置管理

- 功能说明: 提供系统使用级别的配置功能，以树形列表形式展示当前可配置项。
- 关键补充: 用户可直接在界面上进行系统软件功能配置。
- 关键补充: 包括系统各类枚举类型字典管理和系统通用字典管理。
- 关键补充: 支持对通用字典进行新增、修改和删除。
