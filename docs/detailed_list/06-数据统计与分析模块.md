# 数据统计与分析模块

## 模块定位

数据统计与分析是病理科精细化管理的重要组成部分。本模块用于提供详细、准确、可视化且高可阅读性的数据统计与分析能力。

## 子功能清单

### 质控指标统计

- 功能说明: 提供较为常规的数据统计分析报表，用户可通过自定义统计内容输出统计结果表格。
- 关键补充: 系统提供自定义统计管理功能，包括统计参数、适用范围、统计展示形式等。
- 关键补充: 用户选择统计项目名称和统计时间段后，系统自动生成统计数据表格，并支持统计结果 Excel 导出。
- 统计项:

1. 支持三甲评审要求的病理十三项质控指标数据实时统计与一键导出。
2. 支持标本规范化固定率质控、不合格标本及原因质控、临床符合率质控、一线符合率质控、术中快速诊断与石蜡诊断符合率、细胞学病理诊断质控符合率、会诊符合率质控、取消审核报告质控、技术质控、取材工作质量质控、报告实际发放天数质控、标本处理时限质控、术中 / 组织 / 细胞病理诊断及时率及其他自定义质控统计。

### 管理指标统计

- 功能说明: 提供面向科室运营和人员工作量的统计分析能力。
- 统计项:

1. 支持业务量分类统计、收费分析、物资分析、绩效分析等科室运营指标统计。
2. 支持按角色、按业务、多维度的人员工作量统计分析。

### 自定义统计分析

- 功能说明: 提供自定义统计内容和自定义统计报表能力。
- 统计项:

1. 支持自定义统计内容，包括统计参数、适用范围、统计展示形式等。
2. 支持自定义统计报表，选择统计项目名称和统计时间段后自动生成统计数据表格，并支持统计结果 Excel 导出。
