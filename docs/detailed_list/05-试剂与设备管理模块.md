# 试剂与设备管理模块

## 模块定位

病理科需要使用的试剂种类多、规格复杂，病理流程中涉及的仪器设备形式各异，几乎每个病理流程节点都涉及多种设备。本模块用于支持病理科对试剂和设备进行有效管理。

## 子功能清单

### 试剂库存管理

- 功能说明: 用于录入和管理大量试剂库存信息，包括试剂规格、有效期、库存等内容。
- 关键补充: 支持试剂状态标识，便于查看可使用试剂。
- 关键补充: 支持设置订购阈值，并支持按阈值与剩余数量对比和排序。
- 关键补充: 便于科室管理人员直观判断需采购试剂并制定采购计划。

### 试剂信息管理

- 功能说明: 用于维护试剂字典表，不同于库存管理，重点在于管理试剂基础信息。
- 关键补充: 支持录入试剂各方面信息内容并设定提醒阈值。
- 关键补充: 便于科室人员查看相关试剂信息。

### 仪器设备

- 功能说明: 用于登记和管理科室全部需登记的仪器设备类型。
- 关键补充: 支持登记仪器设备的使用情况和维修记录。
- 关键补充: 提供便捷的查询检索功能。
- 关键补充: 通过查询检索记录，可清晰了解设备状态、日常检查、使用情况和维修情况。
