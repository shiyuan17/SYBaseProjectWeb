# 仪表盘

病理看板与个人工作台，聚合当日标本流转、诊断进度与异常指标。

流程：病理看板总览 -> 个人工作台待办 -> 岗位快捷入口

### 1.看板总览与个人待办

用于快速查看全院病理运行态势，并从个人待办直接进入业务工作站。

#### 1.1 病理看板总览

菜单路径：首页 -> 仪表盘 -> 病理看板

角色说明：建议使用管理员账号打开，便于一次性查看完整菜单、筛选条件与维护入口。

操作步骤：

1. 进入病理看板后，先查看左中右三栏的核心指标、待办和异常提醒。
2. 如需核对当日工作量或趋势，可切换时间范围、科室或统计口径。

可选操作：切换看板筛选条件；查看异常指标与待办提醒

预期结果：页面可展示核心卡片、图表和待办区块，适合作为日常值守入口。

#### 1.2 个人工作台与岗位入口

菜单路径：首页 -> 仪表盘 -> 工作台

角色说明：建议使用管理员账号打开，便于一次性查看完整菜单、筛选条件与维护入口。

操作步骤：

1. 打开工作台后，查看当前账号所属岗位的待办列表和快捷入口。
2. 根据待办数量或快捷入口，直接跳转到对应模块继续处理业务。

可选操作：查看个人待办；点击快捷入口进入对应工作站

预期结果：工作台会按岗位聚合待办，支持作为个人日常操作入口。
