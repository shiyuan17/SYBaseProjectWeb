# M6 数据统计与分析

统计仪表盘、质控指标、管理指标、统计报表工作台。

> 本模块截图未捕获。请先运行 `pnpm manual:capture`（需本地联调环境在线）。

## /m6/dashboard

- 主要操作：切换时间维度查看质控、运营与工作量核心指标。

## /m6/quality-indicators

- 主要操作：查看三甲质控、质量安全控制指标与数据源状态。

## /m6/management-indicators

- 主要操作：查看业务量、收费、物资/试剂预警与人员工作量。

## /m6/custom-analysis

- 主要操作：选择分析维度（工作量/质控/冰冻/报告更改/不合格标本）；查询并导出报表。
