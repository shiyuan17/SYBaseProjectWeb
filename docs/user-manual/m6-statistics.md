# M6 数据统计与分析

M6 提供病理质控、运营和工作量分析能力，支撑统计驾驶舱与报表工作台。

流程：统计仪表盘 -> 质控指标统计 -> 管理指标统计 -> 统计报表工作台

### 1.统计驾驶舱

用于快速查看 M6 的核心指标、图表和分析入口。

#### 1.1 统计仪表盘

菜单路径：数据统计与分析 -> 统计仪表盘

角色说明：建议使用统计岗账号打开，确保指标卡片、分析入口和导出操作完整可见。

操作步骤：

1. 进入统计仪表盘后，先查看核心指标卡片和趋势图表。
2. 根据需要切换时间范围或统计维度，快速判断整体运营态势。

可选操作：切换时间维度；查看质控、运营与工作量指标

预期结果：仪表盘会集中展示 M6 关键指标，适合作为统计驾驶舱首页。

### 2.指标与分析工作台

用于查看分专题指标并输出自定义统计分析结果。

#### 2.1 质控指标统计

菜单路径：数据统计与分析 -> 质控指标统计

角色说明：建议使用统计岗账号打开，确保指标卡片、分析入口和导出操作完整可见。

操作步骤：

1. 打开质控指标统计后，查看三甲质控和质量安全控制指标。
2. 如需核对来源，继续检查数据源接入口径和状态信息。

可选操作：查看三甲质控指标；查看质量安全控制指标；核对数据源状态

预期结果：页面应聚合质控类指标和数据源状态，便于质量监控。

截图组：

![质控指标统计：查看三甲质控、质量安全控制指标与数据源状态。](images/m6-statistics/m6-quality-overview.png)

#### 2.2 管理指标统计

菜单路径：数据统计与分析 -> 管理指标统计

角色说明：建议使用统计岗账号打开，确保指标卡片、分析入口和导出操作完整可见。

操作步骤：

1. 进入管理指标统计后，查看业务量、收费、试剂预警和人员工作量。
2. 结合时间范围与指标口径，判断当前运营或资源使用情况。

可选操作：查看业务量；查看收费指标；查看物资预警与人员工作量

预期结果：管理指标统计会聚合运营侧关注的数据口径，便于管理分析。

截图组：

![管理指标统计：查看业务量、收费、物资预警与人员工作量。](images/m6-statistics/m6-management-overview.png)

#### 2.3 统计报表工作台

菜单路径：数据统计与分析 -> 统计报表工作台

角色说明：建议使用统计岗账号打开，确保指标卡片、分析入口和导出操作完整可见。

操作步骤：

1. 进入统计报表工作台后，先选择分析主题和筛选条件。
2. 执行查询后查看结果，并按需要导出工作量、质控、冰冻或不合格标本报表。

可选操作：选择分析维度；查询统计结果；导出报表

预期结果：工作台应支持多种统计专题的统一查询与导出。
