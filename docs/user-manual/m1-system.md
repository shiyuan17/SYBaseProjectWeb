# M1 系统管理

用户、角色、科室/部位字典、医嘱字典/收费/套餐、描写模板、取材规范、系统配置、编号规则、日志。

> 本模块截图未捕获。请先运行 `pnpm manual:capture`（需本地联调环境在线）。

## /system/users

- 主要操作：新增/编辑用户；启停账号；绑定岗位。

## /system/roles

- 主要操作：新增/编辑角色；分配权限码。

## /system/departments

- 主要操作：维护科室组织架构树。

## /system/body-parts

- 主要操作：维护取材部位分类字典。

## /system/medical-order-dicts

- 主要操作：维护医嘱项目基础字典。

## /system/medical-order-charges

- 主要操作：维护收费项目与价格。

## /system/medical-order-packages

- 主要操作：维护常用医嘱套餐组合。

## /system/sampling-templates

- 主要操作：维护大体/镜下/诊断描写模板。

## /system/sampling-guidelines

- 主要操作：维护取材标准操作规范。

## /system/configs

- 主要操作：维护运行参数与开关。

## /system/numbering-rules

- 主要操作：配置病理号/标本号等编号规则。

## /system/logs

- 主要操作：按条件查询操作日志。
