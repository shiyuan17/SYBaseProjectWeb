# M5 归档与借记 / 设备及试剂

归档管理、借记管理、仪器设备、试剂耗材、医疗废物管理。

> 本模块截图未捕获。请先运行 `pnpm manual:capture`（需本地联调环境在线）。

## /operation-support/archive

- 主要操作：查询归档台账；登记/调整归档位置；查看归档明细。

## /operation-support/borrow

- 主要操作：登记借出；办理归还；查看逾期提醒。

## /operation-resources/equipment

- 主要操作：维护设备档案；登记保养记录；查看设备预警。

## /operation-resources/reagents

- 主要操作：维护试剂耗材基础信息；登记库存批次；查看预警。

## /operation-resources/medical-waste

- 主要操作：打印废物袋标签；登记交接记录。
