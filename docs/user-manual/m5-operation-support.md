# M5 归档与借记 / 设备及试剂管理

M5 覆盖归档与借记台账，以及仪器设备、试剂耗材和医疗废物等资源管理能力。

流程：归档管理 -> 借记管理 -> 仪器设备管理 -> 试剂耗材管理 -> 医疗废物管理

### 1.归档与借记

用于维护蜡块、切片、报告的归档台账，以及借出归还和逾期提醒。

#### 1.1 归档管理

菜单路径：归档与借记 -> 归档管理

角色说明：建议使用管理员账号打开，便于一次性查看完整菜单、筛选条件与维护入口。

操作步骤：

1. 进入归档管理后，按病例、档案类型或位置查询目标台账。
2. 根据当前状态维护归档位置，并查看蜡块、切片或报告明细。

可选操作：查询归档台账；登记或调整归档位置；查看归档明细

预期结果：归档管理应完整记录档案位置和变更情况，便于后续借阅或盘点。

#### 1.2 借记管理

菜单路径：归档与借记 -> 借记管理

角色说明：建议使用管理员账号打开，便于一次性查看完整菜单、筛选条件与维护入口。

操作步骤：

1. 打开借记管理后，先查看当前借出、归还和逾期记录。
2. 按实际业务登记借出或归还，并跟踪是否存在超期未还项目。

可选操作：登记借出；办理归还；查看逾期提醒

预期结果：借记管理会保留借出和归还记录，便于查询当前流转状态。

### 2.设备及试剂管理

用于管理仪器设备、试剂耗材和医疗废物等实验室资源。

#### 2.1 仪器设备管理

菜单路径：设备及试剂管理 -> 仪器设备管理

角色说明：建议使用管理员账号打开，便于一次性查看完整菜单、筛选条件与维护入口。

操作步骤：

1. 进入仪器设备管理后，查询当前设备档案与状态。
2. 按需要维护保养记录、查看预警并核对设备生命周期信息。

可选操作：维护设备档案；登记保养记录；查看设备预警

预期结果：设备台账应串联档案、保养与预警信息，便于实验室运维。

#### 2.2 试剂耗材管理

菜单路径：设备及试剂管理 -> 试剂耗材管理

角色说明：建议使用管理员账号打开，便于一次性查看完整菜单、筛选条件与维护入口。

操作步骤：

1. 在试剂耗材管理中查看基础档案、库存批次和当前预警。
2. 按业务需要维护试剂基础信息，并核对批次与余量情况。

可选操作：维护基础信息；登记库存批次；查看预警

预期结果：试剂耗材页面应支撑基础资料维护和库存预警跟踪。

#### 2.3 医疗废物管理

菜单路径：设备及试剂管理 -> 医疗废物管理

角色说明：建议使用管理员账号打开，便于一次性查看完整菜单、筛选条件与维护入口。

操作步骤：

1. 进入医疗废物管理后，先查看废物袋打印与交接记录。
2. 按实际业务打印标签，并登记或回看交接情况。

可选操作：打印废物袋标签；登记交接记录

预期结果：医疗废物管理应保留打印与交接痕迹，便于合规审计。
